> 2026-07-01 | 原则：能用的拿来用，不能用的丢掉

---

## 零、核心定位

**我们做什么：超级编程伙伴** — 能理解你、懂你的习惯、帮你干活
**我们不是什么：不做自研大模型** — 大脑用别人的（DeepSeek/智谱/GPT/Claude），我们做的是伙伴层

```
别人的大模型（DeepSeek/智谱/GPT/Claude...）
        ↓  我们调用
┌──────────────────────────────────┐
│         GBT 超级编程伙伴           │
│  记忆系统 → 推理引擎 → Agent调度    │
│  桌面操控 → 代码执行 → 自进化      │
│  镜像空间 → 安全守卫 → 工具集成    │
└──────────────────────────────────┘
        ↓  伙伴帮你
  写代码 / 修bug / 操控电脑 / 做交易 / 做短剧
```
