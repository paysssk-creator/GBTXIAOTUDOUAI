## 二、开源模式：简洁实用

**模型不用我们的——大脑用别人的，伙伴是我们的。**

```
开源什么：  GBT 超级编程伙伴（框架 + Agent + 记忆 + 桌面操控）
不开源什么：API 密钥、个人配置、钱包私钥

朋友拿到 → pip install → 配自己的 API Key → 就能用
                    ↓
            觉得好用 → 介绍给更多朋友
                    ↓
            朋友要你帮他跑短剧/做策略 → 收个红包
```

**不需要的东西（别浪费时间）：**
- ❌ Open Core 商业授权 → 不搞企业
- ❌ MaaS 云端计费 → 没那么多人用
- ❌ 开源模型权重 → 大脑用别人的
- ❌ SaaS 多租户 → 给朋友用不需要

**需要的东西：**
- ✅ `pip install` 一条命令装好
- ✅ 配置自己的 API Key 就能跑
- ✅ GitHub 公开仓库，MIT 协议
- ✅ README 写清楚怎么用
